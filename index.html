Tentang Saya

Saya adalah mahasiswa S1 di Universitas Teknokrat Indonesia dengan semangat tinggi dalam bidang pengembangan IT, khususnya desain dan pengembangan web. Saya telah membangun fondasi yang kuat dalam menciptakan solusi desain yang berfokus pada pengguna serta merancang situs web yang menarik dan fungsional.

Melalui pengalaman akademis dan proyek freelance, saya terbiasa menggunakan alat-alat standar industri seperti Visual Studio Code (VSCode) dalam proses pengembangan. Fokus saya tidak hanya pada aspek estetika, tetapi juga pada optimalisasi pengalaman pengguna dan kelancaran fungsionalitas situs.

Saya selalu berupaya menyelaraskan konsep desain dengan tujuan proyek agar menghasilkan situs web yang tidak hanya menarik secara visual, tetapi juga efektif dalam memenuhi kebutuhan pengguna. Selain itu, saya memiliki keahlian dalam berbagai perangkat lunak desain dan produktivitas seperti CorelDRAW, Adobe Photoshop, Microsoft Excel, Word, dan PowerPoint.

Dengan keterampilan dan wawasan yang saya miliki, saya sangat antusias untuk berkontribusi dalam proyek-proyek dinamis di dunia desain web yang terus berkembang.
